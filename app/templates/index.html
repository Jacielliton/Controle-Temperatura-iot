Cloud IoT Indústria 4.0 Dashboard

Bem-vindo ao Dashboard da AMA - Associação dos Amigos do Autista do Maranhão.

Esta plataforma foi desenvolvida para coletar, processar e visualizar dados de sensores IoT, facilitando a organização e a utilização dos dados socioeconômicos para melhor tomada de decisões.

Os sensores utilizados incluem:

  • Sensor de Temperatura: Mede a temperatura ambiente (em °C).
  • Sensor de Umidade: Mede a umidade relativa do ar (em %).
